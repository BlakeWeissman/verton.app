Introducing Verton

Verton is a customizable digital assistant currently in development for Windows with a built-in robust command creator that allows you to create and edit commands for Verton.

Download the latest alpha build for Windows (0.3.4)

Digital Assistant

Verton can easily recognize commands using voice or text.

Command Creator

Verton's command creator lets a user easily create or edit commands. The command creator's plethora of features and simple interface allows for a complex, yet easy to use experience.

Settings

Users can easily change Verton's user experience towards their liking in the settings page. Users can also use advanced built-in tools that allows a user to customize Verton at a very low level, like programming custom commands in C# as a more advanced alternative to the command creator. The built-in Verton command compiler also easily lets a user debug their custom code and commands aswell.
